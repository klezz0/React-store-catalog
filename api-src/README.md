Все что от сюда нужно это папка database и app.js

копируем эти файлы в папку с проектом

после копирования этих файлов делаем npm i express и npm i express-fileupload

в файле app.js меняем в перменной process.env.IMAGES_DIR путь куда нужно сохранять картинки

после этого для запуска прописываем node app.js
